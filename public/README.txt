ASR Divine — Public Assets
==========================

Place the following files in this folder:

  logo.png          — Full logo (with dark blue background) — used in Navbar & Footer
                      Recommended size: original file as-is (1052×588 or similar)

  og-image.jpg      — Open Graph / social share image (1200×630 px)
                      Can be the same as logo.png, saved as JPG at that size

  favicon.ico       — Browser tab icon (32×32 px)
                      Generate free at: https://favicon.io/favicon-converter/

  apple-touch-icon.png  — iOS home screen icon (180×180 px)
                          Generate at: https://favicon.io/favicon-converter/

HOW TO SAVE THE LOGO FROM THE CHAT:
1. Right-click the logo image in the Claude chat
2. Click "Save image as..."
3. Save it here as:  logo.png
4. Then save it again as:  og-image.jpg

That's it — the website will automatically pick them up.
